Task 3

User registration form

Create a form similar to the search form for new user registration. Display this form when user clicks on the menu item Create Offer

This form should be visible in the same section as the search form. DO NOT make a full page refresh

The form should have the following fields

  1. First Name (input field name: firstName). Required field. No special characters should be allowed
  2. Last Name (input field name: lastName). Same validations as First name.
  3. Sex (input field name: sex)
  4. Date of Birth (input field name: dob) Required. Make this as user friendly as possible, by either using drop downs for year, month, day or an input mask. Age should be greater than 18 years.
  5. Country (input field name: country) as a select drop down. Populate this field with all countries from which the user can select his/her country

On submit, send an AJAX POST request to http://localhost:3005/users

Display the received response in either a modal dialog or on the same page. It will be in JSON format

  {
  "firstName": "fff",
  "lastName": "lll",
  "sex": "M",
  "dob": "1985-04-12",
  "country": "IN",
